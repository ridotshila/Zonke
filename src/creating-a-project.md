# Creating a Project

After logging into your Zonke account, you can create new projects to manage your work and collaborate with others.

---

## Accessing Project Creation

1. 🖱️ Navigate to the main dashboard
2. 📁 Click the **"New Project"** button in the menu
3. ✏️ The project creation form will open

---

## Project Creation Form

The **Create Project** form allows you to define all the details needed to set up your project.

### Industry Selection

| Field | Description |
|-------|-------------|
| 🏭 Industry | Type to filter industries (e.g., Blockchain, Software Development) or enter industry number (e.g., 4) |

> 💡 **Tip:** Type a name or a number to quickly find your industry category.

### Project Details

| Field | Description | Example |
|-------|-------------|---------|
| 📝 Project Title | Your project name | Zonke Wallet Platform |
| 📄 Project Description | Brief overview of your project | A secure digital wallet solution for users |
| 🔗 GitHub Repository URL | Optional - Link to your project repository | https://github.com/user/repo |

### Project Image

| Option | Description |
|--------|-------------|
| 🖼️ URL | Provide an image URL |
| 📁 Upload | Upload an image file (Size < 5MB) |

> 💡 **Note:** Leave this blank if no new image is being uploaded.

### GitHub Integration

| Field | Description |
|-------|-------------|
| 📂 GitHub Blob URL | Optional - Link to specific file or documentation |

---

## Adding Jobs to Your Project

Each project can have multiple jobs. Click **"+ Add Job"** to add new job listings.

### Job Fields

| Field | Description | Example |
|-------|-------------|---------|
| 📋 Description | Job title or description | Develop Smart Contracts |
| 🔗 Outcome GitHub URL | Optional - Link to expected deliverables | https://github.com/owner/repo/blob/main/path/file.md |
| ⏱️ Hours | Estimated hours for completion | 5 |
| 📅 Deadline | Project completion date | 2024/12/31 |
| 💰 Rate (ADA/hr) | Payment rate in ADA per hour | 10 |

### Job Actions

- ➕ **Add Job** - Add another job to the project
- 🗑️ **Remove** - Remove a job from the list
- 🧹 **Clear** - Clear all job entries

---

## How to Create a Project

### Step 1: Select Industry

1. Click on the Industry field
2. Start typing to filter options (e.g., "Block")
3. Select the appropriate industry category

### Step 2: Enter Project Information

1. 📝 Enter your **Project Title** (e.g., "Zonke Wallet")
2. 📄 Provide a detailed **Project Description**
3. 🔗 Add your **GitHub Repository URL** (optional)

### Step 3: Add Project Image

Choose one of these options:
- 🌐 Paste an image URL
- 📁 Click "Choose File" to upload an image (< 5MB)

### Step 4: Configure GitHub Integration

- 📂 Add **GitHub Blob URL** if you want to link to specific documentation or code files

### Step 5: Add Jobs

1. ➕ Click **"+ Add Job"**
2. 📋 Enter the **Job Description**
3. 🔗 Add **Outcome GitHub URL** (optional)
4. ⏱️ Specify **Hours** for completion
5. 📅 Set the **Deadline** (YYYY-MM-DD format)
6. 💰 Enter the **Rate** in ADA per hour
7. Repeat for additional jobs

### Step 6: Submit Project

After completing all fields:
1. ✅ Review all information for accuracy
2. 🚀 Click **"Submit Project"**
3. ⏳ Wait for confirmation
